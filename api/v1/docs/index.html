CommonDataWealth API Documentation

This API provides access to structured metadata and object identifiers.

⚠️ Public Demo Notice

Important: This API is a public demonstration version. The data stored here is for testing and does not represent actual records.

Available Endpoints

  • Retrieve all metadata: GET /api/v1/metadata
  • Retrieve metadata by ID: GET /api/v1/metadata/{id}
  • Retrieve all object identifiers: GET /api/v1/objects

Versioning

This API follows semantic versioning and is currently at v1.0.0.

Response Format

All responses are returned in application/json format.

Security

As this is a public demo, authentication is not required. The full implementation will include OAuth 2.0 and API key security.

For more details, visit the OpenAPI Specification.
